# Bright-TV
This case study analyzes BrightTV user and viewership data to understand consumption trends and user behavior. Using Databricks (SQL), the data was cleaned, transformed, and enriched through feature engineering such as time zones, viewing periods, age groups, and duration buckets. The final dataset was analyzed in Excel to create visualizations and uncover insights. The final presentation was created using CANVA.

Key findings show that viewership peaks during night-time periods, with young adults driving the majority of consumption. Engagement is largely influenced by content type, time of day, and user demographics. Based on these insights, recommendations were made to optimize content scheduling, target key audiences, and improve engagement during low-usage periods.
